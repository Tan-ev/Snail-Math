# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Initial Release

### Added

* Introduced the `Num` dynamic number type.
* Added support for signed integers, unsigned integers, and floating-point numbers.
* Added constructors for creating `Num` values.
* Added type conversion utilities.

### Arithmetic

* Added addition.
* Added subtraction.
* Added multiplication.
* Added division.
* Added modulo operation.

### Comparison

* Added number comparison.
* Added `min()`.
* Added `max()`.
* Added `clamp()`.

### Number Utilities

* Added `abs()`.
* Added `neg()`.
* Added `sign()`.
* Added average calculation.

### Rounding

* Added `floor()`.
* Added `ceil()`.
* Added `round()`.

### Build System

* Added CMake support.
* Added shared library build support.

### Notes

* **The dynamic number type is the core feature of Snail-Math.**
* **Snail-Math 1.0.0 focuses on providing a simple and flexible numeric interface.**
